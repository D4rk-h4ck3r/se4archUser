# searchUser: Herramienta Python para buscar cuentas de usuario en múltiples plataformas
Descripción
searchUser es una herramienta desarrollada en Python que permite buscar cuentas de usuario en más de 100 sitios web, redes sociales y plataformas en línea. Esta utilidad es útil para aquellos que desean verificar la disponibilidad de un nombre de usuario a través de diferentes servicios.

Características
Busca cuentas de usuario en más de 100 sitios web, redes sociales y plataformas.
Proporciona resultados detallados, indicando si el nombre de usuario está disponible o no.
Fácil de usar a través de la línea de comandos.
Código abierto y personalizable.

Clona el repositorio:
Copiar
$ git clone https://github.com/D4rk-h4ck3r/se4archUser.git

$ cd searchUser

$ pip install -r requirements.txt
Uso
Para buscar cuentas de usuario, ejecuta el script con el siguiente comando:

python searchUser.py <nombre_de_usuario>

El script mostrará los resultados de la búsqueda en los diferentes sitios web, indicando si el nombre existe en la cuentas.
